# test-api

A super simple, very basic, hacked together test api (for testing).

## Endpoints

```
GET /metrics
GET /v1/test/env
GET /v1/test/metadata/task
GET /v1/test/metadata/stats
GET /v1/test/metadata/task/stats
GET /v1/test/mirror
GET /v1/test/panic
GET /v1/test/ping
GET /v1/test/routes
GET /v1/test/status{?code=XXX}
GET /v1/test/version
```
